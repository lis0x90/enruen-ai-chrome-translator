# What is this

I'm just poking around Chrome AI features. 

This extension: 
* loads language translation models for English-to-Russian and Russian-to-English translation.
* provides a simple UI with two text areas to translate text between the two languages using the AI translation model.


# Load the extension in Chrome

1. Open Chrome and go to: chrome://extensions
2. Enable Developer mode (toggle in the top-right corner)
3. Click "Load unpacked"
4. Select the chrome-extension/ folder
5. The extension appears in your toolbar (pin it via the puzzle icon if needed)
